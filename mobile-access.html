üîß Troubleshooting

If you get 404 errors:

  1. Make sure the Django server is running with: python manage.py runserver 0.0.0.0:8000
  2. Check if Windows Firewall is blocking port 8000
  3. Try accessing the debug page first: http://192.168.0.101:8000/debug/

If the site doesn't load at all:

  1. Verify both devices are on the same WiFi
  2. Try pinging from your phone: ping 192.168.0.101
  3. Check your router settings for AP isolation
